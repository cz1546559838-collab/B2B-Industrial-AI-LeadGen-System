# 公开版边界

## 公开版可以包含

- 通用方法论
- 内容矩阵框架
- 运营流程
- AI 提示词思路
- 模板文档
- 虚构模拟案例
- 虚构品牌、虚构客户、虚构数据

## 公开版禁止包含

- 真实公司名称和内部项目名称
- 真实客户资料
- 手机号、微信、邮箱、地址
- 合同、报价、订单、发票、付款信息
- 账号后台截图、投流数据、真实转化数据
- 销售聊天记录、会议原文、内部复盘
- 未公开产品策略、渠道策略、客户名单

## 私有版和公开版区分

- 私有版：可以用于个人整理，但不得上传公开平台。
- 公开版：只能展示方法、流程、模板和模拟案例。
- 私有资料：统一放入 `99_private_not_upload/` 或本项目外的安全目录。

## 发布前检查

1. 是否出现真实公司名。
2. 是否出现真实客户或联系方式。
3. 是否出现真实后台截图或数据。
4. 是否出现合同、报价、订单信息。
5. 是否可以通过细节反推出真实业务。

只要任意一项存在风险，就不得进入公开版。
